# Architectuur- en functionele besluiten

Gebruik voor technische besluiten `ADR-####-korte-titel.md` en voor functionele besluiten `FDR-####-korte-titel.md`.

Een besluit bevat minimaal datum, status, context, besluit, gevolgen en vervangen besluiten. Alleen besluiten met status **Goedgekeurd** mogen een leidend requirement overschrijven.
